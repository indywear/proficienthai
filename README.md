# ProficienThAI

**ProficienThAI** is a Competition-Grade AI Chatbot for Thai Language Learning, designed for Chinese students. It features Adaptive Learning, Gamification, and a Premium Design.

## Features
- **Adaptive AI**: Generates personalized quizzes based on user weaknesses.
- **Smart Nudge**: Teen-style reminders in Chinese to boost engagement.
- **Real-time Dashboard**: Radar charts and progress tracking.
- **Gamification**: Points, Levels, Streaks, and Titles.
- **Premium UI**: Emoji-free, Gold/Dark theme using Flex Messages.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Database**: Supabase (PostgreSQL)
- **AI**: OpenRouter (Claude 3.5 Sonnet)
- **Bot**: LINE Messaging API
- **Styling**: TailwindCSS + Framer Motion
